## 2. 核心业务流程图

### 2.1 完整阅卷业务流程

```mermaid
flowchart TD
    A[教师登录系统] --> B[选择/创建试卷]
    B --> C[上传学生答卷]
    C --> D[系统预处理答卷]
    D --> E{图片质量检查}
    
    E -->|质量良好| F[AI自动阅卷]
    E -->|质量较差| G[手动调整] --> F
    
    F --> H[生成初步评分]
    H --> I{AI置信度检查}
    
    I -->|置信度高| J[自动确认评分]
    I -->|置信度低| K[人工复核]
    
    K --> L[调整评分]
    L --> J
    J --> M[生成最终成绩]
    M --> N[成绩统计分析]
    N --> O[生成学情报告]
    O --> P[推送结果给教师]
    
    style A fill:#ff6b6b
    style F fill:#4ecdc4
    style O fill:#45b7d1
```
